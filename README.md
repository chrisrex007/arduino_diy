# Arduino_DIY

## Project Title : Bluetooth Controll & Obstacle Avoiding Car

### Components Used: 
- Arduino UNOR3
- L298N Motor Driver
- TT Gear Motor
- 18650 Battery
- 2S, 18650 Battery Holder
- 65MM Wheels for TT Motors
- HC-05 Bluetooth
- SG 90 Servo Motor
- Jumper Wires
- HC-SR04 Ultrasonic Sensor
